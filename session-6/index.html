Part 2 · Technical SEO Audit

Critical Technical Issues

Findings from auditing planeterra.org using Google Search Console, PageSpeed Insights, Screaming Frog, and WooRank.

!

No SEO Plugin Installed

WordPress site has no Yoast, RankMath, or equivalent. This means no automated meta tags, sitemaps, schema, or on-page analysis.

!

Missing Meta Descriptions

Key pages (About, Our Work, Projects) have no meta descriptions. Google auto-generates snippets — no control over SERP appearance.

!

No Structured Data / Schema Markup

No Organization, Article, or FAQ schema detected. Missing out on rich snippets in search results.

~

Generic Title Tags

Using "[Page] - Planeterra" format with no keywords. Should include target terms like "community tourism."

~

No Canonical Tags

Risk of duplicate content issues. WordPress may create multiple URLs for the same content without canonical directives.

~

Weak Robots.txt

Minimal configuration with no sitemap reference and no crawl directives for search engines.

~

No Open Graph Tags

Social media previews are uncontrolled when URLs are shared on Facebook, LinkedIn, or Twitter.

~

Spanish Pages Missing Hreflang

Spanish content exists but lacks hreflang tags, hurting multilingual SEO and causing potential cannibalization.

Technical Recommendations

Priority Action Plan

Ranked by impact and urgency. The #1 fix alone resolves ~50% of the issues found.

Priority 1

Install SEO Plugin

Install Yoast or RankMath to instantly get: meta descriptions, canonical tags, XML sitemaps, schema markup, and on-page scoring for every page.

Fixes 8+ issues at once
Priority 2

Optimize Title Tags

Rewrite all page titles to include target keywords. Example: "Community Tourism Foundation | Planeterra" instead of "Home - Planeterra".

~2 hours of work
Priority 3

Add Schema Markup

Implement Organization, Article, and FAQ structured data to enable rich snippets in Google — sitelinks, knowledge panel, and featured answers.

Enables rich results

Website Navigation Structure

Current structure is well-organized. Recommendation: flatten key conversion pages (Donate, Get Involved) to be no more than 1 click from homepage.

planeterra.org (Home)
→ /about → /our-team → /financials
→ /our-work → /projects → /community-network
→ /blog → /guide/community-tourism (NEW)
→ /donate → /get-involved → /learning-hub

Backlinking Recommendations

Link Building Strategy

Actionable recommendations to strengthen Planeterra's off-page SEO and domain authority.

Guest Posting

Travel Media Outreach

Pitch guest articles to Lonely Planet, National Geographic Travel, BBC Travel, and Condé Nast. Topics: "Community Tourism in 2026" or "How Travel Can Fight Poverty."

Target: 5-10 high-DA links
Academic

University Partnerships

Partner with tourism and sustainability programs at universities. Offer case study data, guest lectures, and student project opportunities. Each yields a .edu backlink.

Target: 3-5 .edu links
Digital PR

Data-Driven Content

Publish original research: "State of Community Tourism 2026" report with statistics and infographics. Journalists and bloggers naturally link to data sources.

Target: 20+ organic links

Quick Wins

  • Claim and optimize Google Business Profile
  • Submit to HARO (Help A Reporter Out) for travel topics
  • Get listed on resource pages of eco-travel directories
  • Request backlinks from existing partners (tour operators)
  • Fix broken links pointing to old Planeterra URLs

Long-Term Strategy

  • Build "Community Tourism Resource Hub" as linkable asset
  • Launch annual impact report with embeddable infographics
  • Sponsor sustainability conferences for event page links
  • Create ambassador program with bloggers/influencers
  • Pursue Wikipedia page for "Community Tourism" with Planeterra reference
